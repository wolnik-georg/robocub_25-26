Event and state notifications¶

You may receive notifications from NAO system or from an application.

In that case, a small sound is played and the status LEDs remain still white but blink once in a while with a colour indicating the severity of the notification.

Status LEDs is ... Description

Steady White light,

cut by a brief

blinking in ...

 Green NAO gives you some feed-back by an Info notification about an ongoing action.
Yellow

Warning notification occurs when an issue requires your attention and/or action.

NAO is still usable but, it may either run in a degraded mode in which some features may be not available or it may soon become unusable if you don’t solve the problem.
Red

Error notification occurs when one or all functionality(ies) of NAO become unusable.

Try to solve the problem using the tips. If the problem persists, contact the customer support and provide them with the error ID. For further details, see: Contacting customer support for NAO.

Download a printable version: Status LEDs - Cheat sheet.

To ask NAO to read the notification, see: Requesting technical information.
