CMake wrapper for swig / Java¶

qi_swig_wrap_java(module_name interface_file [PACKAGE <package>]
    [OUTDIR <outdir>]
    [CPP]
    [SRC <src> ...]
    [DEPENDS <depends> ...]
)
¶
Arguments:
  • module_name – the target name
  • interface_file – the swig interface file (extension is .i)
  • PACKAGE – package name
  • OUTDIR – swig output directory
  • CPP – whereare the lib is in C++
  • SRC – The list of source files
  • DEPENDS – The list of dependencies

Create a java wrapping of the C/C++ library

Warning

The name must match the name declared in interface_file for instance, if name equals Foo, Foo.i must contain: %module Foo, and the generated library will be called ‘libFoo.so’
