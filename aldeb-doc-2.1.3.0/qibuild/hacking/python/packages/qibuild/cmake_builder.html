qibuild.cmake_builder - Building CMake projects¶

CMakeBuilder¶

class qibuild.cmake_builder.CMakeBuilder(build_worktree, projects=[])¶

CMake driver. Allow building multiples cmake projects together. Dependencies can optionally be resolved and be taken into account.

Add a project to the list of projects
