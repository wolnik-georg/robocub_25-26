qisrc.git_config – Handling git configurations¶

qisrc.worktree.Remote¶

class qisrc.git_config.Remote¶

A remote, as stored in the .qi/git.xml file

name¶

The name of the remote

url¶

The url of the remote Note that in case of a gerrit repository, this will be the public, http:// url, qisrc push making sure we use the ssh url when uploading patch sets.

review¶

Wether the remote supports code review.

qisrc.worktree.Branch¶

class qisrc.git_config.Branch¶
name¶

The name of the branch

tracks¶

The name of the remote tracked by this branch

remote_branch¶

The name of the remote branch tracked by this branch. Default is a remote with the same name

default¶

Wether this is the default branch. qisrc sync, will try to synchronize this branch by default
