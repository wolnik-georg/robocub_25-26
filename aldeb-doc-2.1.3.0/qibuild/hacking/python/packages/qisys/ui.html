qisys.ui – Tools for the command line user interface¶

Output messages for the user¶

qisys.ui.error(*tokens, **kwargs)¶

Print an error message

qisys.ui.warning(*tokens, **kwargs)¶

Print a warning message

qisys.ui.info(*tokens, **kwargs)¶

Print an informative message

qisys.ui.debug(*tokens, **kwargs)¶

Print a debug message

You can use each of this functions as you would do for the print function in Python3.

To add colors, use the various colors defined in qisys.ui

Example

from qisys import ui

ui.info(ui.green, "*", ui.bold, "(1/1)", ui.blue, project_name)
