NAOqi PeoplePerception¶

This section describes the modules that can be used to analyse the people around the robot.

All robots¶

Use ... To ...
ALBasicAwareness Make the robot react to its environment’s stimuli.
ALEngagementZones Analyze the position of a person with respect to the robot.
ALFaceCharacteristics Provide estimated human characteristics based on facial analysis.
ALFaceDetection Make the robot detect and recognize human faces.
ALGazeAnalysis Analyze the direction of the gaze of a person.
ALPeoplePerception Make the robot keep tracks about the people around it.
ALSittingPeopleDetection Make the robot detect whether a person is sitting or not.

Requiring a 3D sensor¶

Use ... To ...
ALWavingDetection Make the robot detect if a detected person is waving.
